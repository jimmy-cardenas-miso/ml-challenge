## 🚀 Getting Started

### Frontend

```sh
   cd frontend
   npm install
```
### Backend

```sh
   cd backend
   npm install
```

## ✨ Run project

### Frontend

```sh
   cd frontend
   npm run dev
```
```sh
  open url: http://localhost:5173/
```


### Backend

```sh
   cd backend
   npm run start:app
```
```sh
  open url: http://localhost:8080/
```

## 🔍 Run linters
```sh
   cd frontend
   npm run lint
```

## 📝 Run test
```sh
   cd frontend
   npm run test
```

### Coverage

```sh
   cd frontend
   npm run coverage
```
